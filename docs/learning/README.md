# Aprendizaje — movido fuera de cosmos

El playbook y las extracciones viven en un repo/carpeta **personal**, no en este
proyecto:

**`../engineering-playbook/`**  
(ruta completa: `Projects/engineering-playbook/`)

| Qué | Dónde |
|-----|--------|
| Reglas vivas | `engineering-playbook/MY-PLAYBOOK.md` |
| Extracción Phase 0 | `engineering-playbook/learnings/cosmos/LEARN-phase-0-portable-2026-06-12.md` |
| Skill Cursor | `learning-playbook` en `~/.cursor/skills/` |

## Uso

- **Antes de un proyecto nuevo** → abrí `MY-PLAYBOOK.md`
- **Después de un gate en cosmos** → *"Extracción portable Phase N desde cosmos, guardar en engineering-playbook"*
- **Actualizar reglas** → editá `MY-PLAYBOOK.md` o pedí al skill que promueva un patrón

Cosmos queda como laboratorio; el aprendizaje portable es transversal.
